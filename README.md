# DecodeLabs-Internship
TASK 01- Student Survival Hub
 is a responsive web application designed to help students manage their academic life efficiently. It includes tools such as a CGPA calculator, assignment tracker, Pomodoro timer, exam countdown, notes organizer, and to-do list, all within a clean and user-friendly interface.



📖 About the Project

Student Survival Hub is a responsive web application designed to help students organize and manage their academic activities in one place. It provides essential productivity tools to improve time management, study planning, and overall academic performance through a clean and user-friendly interface.
